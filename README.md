# Snake-Javascript
Juego de snake basado en el video: https://www.youtube.com/watch?v=xBVYyto4U5Y

## Aclaraciones
La administracion del puntaje se realiza a traves de la clase juego y su atributo puntaje. La clase posee un metodo aumentar que suma uno al puntaje, un metodo reiniciar que lo pone en cero y un metodo get que retorna el puntaje.
